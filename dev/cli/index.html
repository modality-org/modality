CLI Reference

The modal command-line tool is your interface to Modality.

Command Overview

CommandDescription
modal contractContract management (create, commit, push, pull)
modal idIdentity management (create, derive, get)
modal predicatePredicate management and testing
modal nodeNetwork node operations
modal hubHub server management

Contract Commands (modal contract / modal c)

Create a Contract

modal contract create
modal c create

Creates a new contract in the current directory.

Checkout State

modal c checkout

Extracts current state to state/ for editing.

Set State Values

modal c set /path/to/file.txt "content"
modal c set /data/config.json --file ./local.json

Commit Changes

# Commit all changes
modal c commit --all -m "Description"

# Commit with signature
modal c commit --all --sign alice.passfile -m "Signed"

# Commit an action
modal c commit --action '{"type":"DEPOSIT"}' --sign alice.passfile

View History

modal c log
modal c log --verbose

Check Status

modal c status

Push/Pull

# Push to hub
modal c push http://hub.example.com/contracts/my-contract

# Pull from hub
modal c pull http://hub.example.com/contracts/my-contract

Identity Commands (modal id)

Create Identity

modal id create --path alice.passfile

Get Public ID

modal id get --path alice.passfile
# Output: ed25519:abc123...

Derive Sub-Identity

modal id derive --path alice.passfile --sub "escrow" --output alice-escrow.passfile

Predicate Commands (modal predicate)

List Predicates

modal predicate list

Get Predicate Info

modal predicate info signed_by
modal predicate info threshold

Node Commands (modal node)

Create a Node

modal node create --path ./my-node

Start/Stop

modal node start --path ./my-node
modal node stop --path ./my-node

View Info

modal node info --path ./my-node
modal node address --path ./my-node

Common Patterns

Create and Initialize

mkdir my-contract && cd my-contract
modal c create
modal id create --path alice.passfile
modal id create --path bob.passfile
modal c checkout
modal c set /parties/alice.id $(modal id get --path ./alice.passfile)
modal c set /parties/bob.id $(modal id get --path ./bob.passfile)
modal c commit --all --sign alice.passfile -m "Initial setup"

Execute an Action

modal c commit --action '{"type":"DEPOSIT"}' --sign alice.passfile
