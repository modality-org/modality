# @modalmoney/sdk

Modal Money JavaScript SDK for web applications.

## Installation

```bash
npm install @modalmoney/sdk
```

or

```bash
pnpm add @modalmoney/sdk
```

## Usage

```javascript
import { version } from '@modalmoney/sdk';

console.log('Modal Money SDK version:', version);
```

## Development

This package is part of the Modal Money monorepo.

### Testing

```bash
pnpm test
```

### Building

```bash
pnpm build
```
