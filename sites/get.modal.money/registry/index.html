Build from Source (Cargo)

To build and install from this registry:

cargo install --index sparse+http://packages.modality.org/testnet/latest/cargo-registry/index/ modality

Registry Configuration

Add to ~/.cargo/config.toml:

[registries.modality]
index = "sparse+http://packages.modality.org/testnet/latest/cargo-registry/index/"

Then install with:

cargo install --registry modality modality

About

This registry contains all Modality workspace crates with proper dependency resolution. When you install modality, all internal dependencies will be automatically fetched from this registry.
