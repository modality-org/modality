# Modal - Main Landing Page

This directory contains the main landing page for `modal.money`.

## Overview

The modal.money landing page presents the core value proposition: **"Money That Rules"** and directs users to get.modal.money for installation and downloads.

## Structure

```
sites/modal.money/
├── index.html      # Main landing page
└── README.md       # This file
```

## Deployment

This page should be deployed to the root of modal.money domain.

The site includes:
- Minimal, impactful design with the tagline "Money That Rules"
- Clear call-to-action button linking to get.modal.money
- Responsive design for mobile and desktop

## Domain Configuration

- **Main Site**: `https://modal.money` - Landing page with value prop
- **Download Site**: `https://get.modal.money` - Installation and package downloads
